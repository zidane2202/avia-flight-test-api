# avia-flight-test-api

API de test pour le workflow Avia (Ease Travel) — recherche de vols multi-agences.

## Endpoints

- `GET /health`
- `GET /v2/bot/place?text=`
- `GET /v2/bot/search?...`
- `GET /v2/bot/agencies`
- `GET /v2/bot/agency?name=`
- `GET /staff/notify?reason=&summary=&sessionId=`

## Local

```bash
npm install
npm start
```
